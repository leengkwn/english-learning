# 🚀 快速開始指南

## 📥 第一次使用

### 1. 解壓縮檔案
將 `小學英文學習工具包_20251207_v3.0.zip` 解壓縮到任意資料夾

### 2. 開啟首頁
雙擊 `學習中心首頁_20251207_v1.4.html`

### 3. 開始學習!
從首頁選擇想使用的工具

---

## 🎮 各工具快速使用

### 📖 單字閃卡系統
1. 選擇單元 (Unit 1-10)
2. 點擊單字卡查看詳細資訊
3. 使用🔊按鈕聽發音
4. 點「下一個」繼續學習

**快捷鍵:**
- 空白鍵: 翻卡
- → 方向鍵: 下一張
- ← 方向鍵: 上一張

---

### 🧩 單字拼字遊戲
1. 看中文提示
2. 拼出英文單字
3. 使用提示按鈕 (字母數量/首字母/完整答案)
4. 輸入完成後按Enter或點「檢查」

**提示:**
- 先不看提示試著拼
- 卡住時再用提示
- 全對會有特效!

---

### 🎴 單字配對遊戲
1. 選擇難度 (4x4簡單 / 4x6困難)
2. 點擊卡片翻開
3. 找到配對的英文和中文
4. 挑戰最短時間記錄!

**技巧:**
- 記住已翻開卡片的位置
- 先找簡單的配對
- 多練習提升記憶力

---

### 📄 句子拼圖遊戲 v3.0 ⭐

#### 步驟1: 選擇難度
- 😊 **簡單** - 小一~小三,基礎句型
- 🤔 **中等** - 小三~小五,進階句型
- 😤 **困難** - 小五~小六,複雜句型

#### 步驟2: 拼句子
1. 看中文提示
2. 拖曳或點擊單字到答案區
3. 排列成正確順序
4. 注意標點符號!

#### 步驟3: 檢查答案
- 答對 ✅ → 自動語音朗讀 → 下一題
- 答錯 ❌ → 顯示正確答案 → 再試一次

#### 步驟4: 完成後
- 🎮 **再玩一次** - 相同難度,全新10題
- 🏠 **回首頁** - 換個難度試試

## 💡 學習策略

### 🎯 初學者 (小一~小二)
```
第1週: 閃卡 Unit 1-2 → 配對遊戲 4x4
第2週: 複習 + 簡單句子拼圖
第3週: 閃卡 Unit 3 + 繼續練習
第4週: 總複習所有工具
```

### 📚 中階學習者 (小三~小四)
```
每天20分鐘:
- 閃卡複習 (5分鐘)
- 句子拼圖中等難度 (10分鐘)
- 拼字或配對遊戲 (5分鐘)
```

### 🏆 進階學習者 (小五~小六)
```
挑戰模式:
- 困難句子拼圖全對
- 拼字遊戲不用提示
- 配對遊戲最短時間
```

## 🎯 常見問題

### Q: 需要網路嗎?
**A:** 不需要!所有工具都可以離線使用。

### Q: 可以在手機上用嗎?
**A:** 可以!所有工具都支援手機、平板、電腦。

### Q: 進度會保存嗎?
**A:** 會!進度保存在瀏覽器的本地儲存中。
⚠️ 注意:清除瀏覽器資料會刪除進度。

### Q: 句子拼圖每次題目都一樣嗎?
**A:** 不會!每次隨機從20題中抽10題,每次都不同。

### Q: 如何分享給學生?
**A:** 
1. 整個資料夾複製給學生
2. 或上傳到學校網路硬碟
3. 或用USB隨身碟分享

### Q: 可以投影在教室螢幕嗎?
**A:** 可以!非常適合課堂教學使用。
