### 📄 5. 句子拼圖遊戲 v3.0 ⭐新版
**檔案:** `句子拼圖遊戲_20251207_v3.0.html`
**大小:** 45KB

**三級難度系統:**

#### 😊 簡單難度 (50-100單字)
- **適合:** 小一~小三
- **題數:** 20題題庫,每次10題
- **句型:**
  - 基礎be動詞句型
  - 簡單現在式
  - 基本疑問句
  - 3-5個單字短句

**範例題目:**
```
我是學生。 → I am a student.
你好嗎? → How are you?
他喜歡狗。 → He likes dogs.
```

#### 🤔 中等難度 (100-250單字)
- **適合:** 小三~小五
- **題數:** 20題題庫,每次10題
- **句型:**
  - 現在進行式
  - 介系詞片語
  - 動名詞用法
  - 過去式
  - 5-7個單字句子

**範例題目:**
```
我每天上學。 → I go to school every day.
她正在看書。 → She is reading a book.
他們在公園玩。 → They play in the park.
```

#### 😤 困難難度 (250-400單字)
- **適合:** 小五~小六
- **題數:** 20題題庫,每次10題
- **句型:**
  - 比較級/最高級
  - 條件句
  - 完成式
  - 複合句型
  - 7-10個單字長句

**範例題目:**
```
我昨天在公園遇見我的老師。
→ I met my teacher in the park yesterday.

他比我高。
→ He is taller than me.

如果明天下雨,我們就待在家。
→ If it rains tomorrow, we will stay home.
```

**核心功能:**
- 🎯 三級難度自由選擇
- 🔀 每次隨機抽題,保持新鮮
- 🎮 再玩一次 (相同難度,全新10題)
- 🏠 回首頁 (重新選擇難度)
- 🔊 答對自動語音朗讀
- 📱 全平台支援 (電腦/平板/手機)
- ✋ 拖曳/點擊雙操作模式

## 💡 使用建議

### 課堂教學 (40分鐘)
```
前10分鐘: 閃卡導入新單字
中20分鐘: 句子拼圖分組競賽
後10分鐘: 配對遊戲複習
```

### 課後練習 (30分鐘/天)
```
閃卡複習: 10分鐘
句子拼圖: 15分鐘 (依能力選難度)
自選遊戲: 5分鐘
```

### 補救教學
```
1. 診斷: 句子拼圖找出弱點
2. 加強: 針對性閃卡練習
3. 鞏固: 配對/拼字遊戲
```

### 個別化學習
```
學生A (程度較弱):
  閃卡Unit 1-3 → 簡單句子拼圖 → 配對4x4

學生B (程度中等):
  閃卡Unit 1-6 → 中等句子拼圖 → 拼字遊戲

學生C (程度較好):
  閃卡Unit 1-10 → 困難句子拼圖 → 全工具挑戰
```
